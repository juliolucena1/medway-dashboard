# 🧠 MEDWAY Analytics Dashboard

Sistema de monitoramento de consultas psicológicas em tempo real.

## 🚀 Funcionalidades

✅ Dashboard em tempo real (30s)  
✅ 4 métricas principais  
✅ Sistema de alertas automáticos  
✅ Exportação CSV  
✅ Design responsivo  

## 🛠️ Tecnologias

- Next.js 14
- TypeScript  
- Tailwind CSS
- Supabase
